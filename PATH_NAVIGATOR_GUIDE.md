## 1. 座標系の定義

### 格子の3軸

```
  v (仰角)
  ↑
6 │  ← +60° 鳥瞰
5 │  ← +45°
4 │  ← +30°
3 │  ← +15°
2 │  ← 0°  水平（目線）★基本
1 │  ← -15°
0 │  ← -30° ローアングル
  └──────────────────────── t (時間)
  0   1   2   3   4   5   6   7   8
  卵 幼虫 蛹  成虫 ...（GLBステージ番号）

  h (水平角) は上から見た俯瞰図:
       h=0 (0°, 正面)
          ↑
  h=54 ←  ● → h=18  (●=被写体)
  (270°)   ↓   (90°)
        h=36 (180°, 真後ろ)

  ※ grid_theta=72 の場合: 1ステップ = 5°
     h=0:0°  h=9:45°  h=18:90°  h=27:135°
     h=36:180°  h=45:225°  h=54:270°  h=63:315°
```

### 前提設定（本ガイドの例）

| 設定 | 値 | 意味 |
|------|----|------|
| `n_stages` | 9 | GLBファイル数 → t = 0〜8 |
| `grid_theta` | 72 | 水平分割数 → h = 0〜71、1ステップ=5° |
| `grid_elev` | 7 | 仰角段階数 → v = 0〜6 |
| 1 h ステップ | 5° | 360° ÷ 72 |
| 1 v ステップ | 15° | (-30°〜+60°) ÷ 6段階 |

> **あなたの設定に合わせた換算**:
> - 1hステップの角度 = 360 ÷ grid_theta
> - 1vステップの角度 = 90 ÷ (grid_elev - 1) （-30°〜+60°の場合）

## 2. 入力フォーマット

### PathNavigator の `path_custom` 入力

```
# コメント行 (# で始まる行は無視)
t, h, v        ← 整数インデックス形式（推奨）
t.t, h.h, v.v  ← 正規化float形式 (0.0〜1.0)
```

**2つの書き方**:

```
# 整数インデックス形式
0, 0, 2       # t=0(stage0), h=0(正面), v=2(水平0°)
1, 9, 2       # t=1(stage1), h=9(45°), v=2(水平0°)

# 正規化float形式 (0.0〜1.0)
0.0, 0.0, 0.333    # 同じ意味 (0.333 ≈ 2/6 = v=2)
0.125, 0.125, 0.333
```

> 混在は可能。小数点が含まれる行はfloat形式として解釈される。

## 3. 基本パターン集

### ─── 時間軸の操作 ─────────────────────────────────────────

#### 【時間正順】ステージを順番に進める（最もシンプル）

カメラを正面(h=0)・水平(v=2)に固定し、時間だけ進める。

```
# 正面固定・時間を0→8まで順に
0, 0, 2
1, 0, 2
2, 0, 2
3, 0, 2
4, 0, 2
5, 0, 2
6, 0, 2
7, 0, 2
8, 0, 2
```

> **クリップ数**: 8クリップ
> **映像**: カメラ固定のまま被写体が成長する

---

#### 【時間逆順】ステージを逆に戻る

```
# 成虫から卵へ戻る（逆再生）
8, 0, 2
7, 0, 2
6, 0, 2
5, 0, 2
4, 0, 2
3, 0, 2
2, 0, 2
1, 0, 2
0, 0, 2
```

> LTX-2 は時間軸の方向に制約がないため、逆行も自然に補間される

---

#### 【時間往復】行って返ってくる

```
# 0→4→0 とUターン
0, 0, 2
1, 0, 2
2, 0, 2
3, 0, 2
4, 0, 2
3, 0, 2
2, 0, 2
1, 0, 2
0, 0, 2
```

---

#### 【時間停止】特定のステージに長く留まる

同じtを繰り返して水平角を変えると、ステージを止めたまま周回できる。

```
# stage3(蛹)で一周してから次へ
3,  0, 2
3,  9, 2
3, 18, 2
3, 27, 2
3, 36, 2
3, 45, 2
3, 54, 2
3, 63, 2
4,  0, 2    # ← ここで時間が1進む
```

---

### ─── 水平角の操作 ─────────────────────────────────────────

#### 【右回り半周】正面から背面まで右回りで移動

```
# h=0(正面) → h=36(背面) を右回りで
0,  0, 2
1,  9, 2    # +45°
2, 18, 2    # +90°
3, 27, 2    # +135°
4, 36, 2    # +180° 背面
```

> 各ステップで時間も1進んでいる（斜め移動）

---

#### 【右回り一周】正面から一周して正面に戻る

```
# h を0→71→0 まで8ステップで一周 (grid_theta=72)
# 1ステップあたり h += 9 (= 72÷8 = 45°ずつ)
0,  0, 2    # 0°
1,  9, 2    # 45°
2, 18, 2    # 90°
3, 27, 2    # 135°
4, 36, 2    # 180°
5, 45, 2    # 225°
6, 54, 2    # 270°
7, 63, 2    # 315°
8,  0, 2    # 360°=正面に戻る
```

---

#### 【左回り一周】正面から逆方向に一周

```
# h を0→-9(=63)→-18(=54)... と左回り
0,  0, 2
1, 63, 2    # -45°
2, 54, 2    # -90°
3, 45, 2    # -135°
4, 36, 2    # -180°
5, 27, 2    # -225°
6, 18, 2    # -270°
7,  9, 2    # -315°
8,  0, 2    # 戻る
```

---

#### 【振り子】正面を中心に左右に揺れる

```
# 正面(h=0) を中心に ±45°(9ステップ) で揺れる
0,  9, 2    # +45°(右)
1,  0, 2    # 正面
2, 63, 2    # -45°(左、72-9=63)
3,  0, 2    # 正面
4,  9, 2    # +45°(右)
5,  0, 2    # 正面
6, 63, 2    # -45°(左)
7,  0, 2    # 正面
8,  9, 2    # +45°(右)
```

---

#### 【横顔固定】特定の角度に固定して時間だけ進める

```
# 右横顔(90°)固定 (h=18, grid_theta=72)
0, 18, 2
1, 18, 2
2, 18, 2
3, 18, 2
4, 18, 2
5, 18, 2
6, 18, 2
7, 18, 2
8, 18, 2
```

---

### ─── 仰角の操作 ─────────────────────────────────────────

#### 【上昇】ローアングルから鳥瞰へ

```
# v=0(-30°)から v=6(+60°)まで上昇しながら時間進行
0, 0, 0    # stage0, 正面, -30°(ローアングル)
1, 0, 1    # stage1, 正面, -15°
2, 0, 2    # stage2, 正面, 0°(水平)
3, 0, 3    # stage3, 正面, +15°
4, 0, 4    # stage4, 正面, +30°
5, 0, 5    # stage5, 正面, +45°
6, 0, 6    # stage6, 正面, +60°(鳥瞰)
7, 0, 6    # stage7, 鳥瞰のまま
8, 0, 6    # stage8, 鳥瞰のまま
```

---

#### 【下降】鳥瞰からローアングルへ

```
# v=6(+60°)から v=0(-30°)へ下降
0, 0, 6    # 鳥瞰から開始
1, 0, 5
2, 0, 4
3, 0, 3
4, 0, 2    # 水平(目線)
5, 0, 1
6, 0, 0    # ローアングル
7, 0, 0
8, 0, 0
```

---

#### 【仰角振り子】水平↔鳥瞰を往復

```
# v: 2(水平)→6(鳥瞰)→2(水平)→0(ロー)→2(水平)
0, 0, 2    # 水平
2, 0, 4    # 上昇中
4, 0, 6    # 鳥瞰
6, 0, 4    # 下降中
8, 0, 2    # 水平に戻る
```

> ウェイポイントは等間隔でなくてもOK (t=0,2,4,6,8 のように飛ばせる)
> ただし |Δt|=2 なので品質低下の可能性あり

### ─── 複合パターン ────────────────────────────────────────

#### 【螺旋上昇】右回りしながら上昇（空へ舞い上がる視点）

# t, h, v
# 右回り (h++) しながら上昇 (v++) ×4ステップ

2, 8, 5
2, 9, 5
3, 9, 6
```

---

#### 【螺旋下降】鳥瞰から降りながら時間進行

```
0,  0, 6    # 鳥瞰・正面
1,  9, 5    # 右45°・+45°
2, 18, 4    # 右90°・+30°
3, 27, 3    # 右135°・+15°
4, 36, 2    # 右180°・水平
5, 45, 2    # 右225°・水平
6, 54, 2    # 右270°・水平
7, 63, 2    # 右315°・水平
8,  0, 2    # 正面・水平
```

---

#### 【ドラマチック登場】ローアングル正面→水平右回り周回

```
# フェーズ1: ローアングルで登場（stage0で仰角上昇）
0, 0, 0    # 地面レベル
0, 0, 1
0, 0, 2    # 水平に上がる
# フェーズ2: 水平を保ちながら右回り
1, 9,  2
2, 18, 2
3, 27, 2
4, 36, 2
5, 45, 2
6, 54, 2
7, 63, 2
8,  0, 2   # 正面に戻る
```

---

#### 【調査飛行】真上から降下しながら一周

```
# 真上(v=6)から始まり水平に降下しながら時間進行
0,  0, 6    # 真上・stage0
2,  9, 5
4, 18, 4
6, 27, 3
8, 36, 2    # 水平・背面・stage8
```

---

#### 【時間往復 + 旋回】前半で時間を進め、後半で逆戻り

```
# 前半: t=0→4, h=0→36 (正面→背面, 右回り)
0,  0, 2
1,  9, 2
2, 18, 2
3, 27, 2
4, 36, 2    # 背面・stage4
# 後半: t=4→8, h=36→72=0 (背面→正面, さらに右回り継続)
5, 45, 2
6, 54, 2
7, 63, 2
8,  0, 2    # 正面に戻る・stage8
```

---

#### 【過去回帰】最新から最古へ逆戻りしながら上昇

```
# 時間を逆行させながら仰角を上げる（独特な演出）
8, 0, 2    # 成虫・水平
7, 0, 3
6, 0, 4
5, 0, 4
4, 0, 5
3, 0, 5
2, 0, 6
1, 0, 6
0, 0, 6    # 卵・鳥瞰

## 7. 隣接制約について

### 推奨: 各ステップで |Δt|≤1, |Δh|≤1, |Δv|≤1

LTX-2 は「2枚の画像の間を AI で動画補間」する。
差が大きいほど別物に見えてしまう可能性がある。

### 大きく移動したい場合 → 中間ウェイポイントを追加

```
# ❌ h が一気に0→18(90°)ジャンプ
0, 0, 2
4, 18, 2

# ✅ 中間を挟む（h=6ずつ、t=1ずつ進める）
0,  0, 2
1,  6, 2
2, 12, 2
3, 18, 2
```

```
# ❌ v が一気に 2→6 ジャンプ
0, 0, 2
4, 0, 6

# ✅ 1段ずつ上昇
0, 0, 2
1, 0, 3
2, 0, 4
3, 0, 5
4, 0, 6
```

### 違反しても動作は続く

`check_adjacency()` で警告が出るが実行は継続される。
仕上がりを見て許容できれば問題ない。

## 8. ウェイポイント設計のコツ

### 「何を見せたいか」から逆算する

```
1. 見せ場のフレームを決める
   例: stage3(蛹→羽化)を正面(h=0)・水平(v=2)でじっくり見せたい

2. 見せ場の前後をどう移動するか決める
   前: ローアングルから上昇しながら近づく
   後: 右回りしながら遠ざかる

3. ウェイポイントを書く
   # 前半: ローアングルから上昇
   0, 0, 0
   1, 0, 1
   2, 0, 2
   # 見せ場: stage3を正面で止まって観察（水平を維持）
   3, 0, 2
   # 後半: 右回りしながら遠ざかる
   4, 9, 2
   5, 18, 3
   6, 27, 4
   7, 36, 5
   8, 45, 6
```

### 整数より正規化floatが便利なケース

```
# n_stages=5, grid_theta=36, grid_elev=5 など変わった値でも
# 比率で書けるので計算不要

# 正規化float (0.0〜1.0)
0.0, 0.0,   0.5      # 最初のstage, 正面, 中間仰角
0.25, 0.25, 0.5      # 1/4進んだstage, 1/4右回転
0.5,  0.5,  0.75     # 中間stage, 半周, 3/4の高さ
0.75, 0.75, 1.0      # 3/4stage, 3/4周, 最高仰角
1.0,  1.0,  1.0      # 最終stage, ほぼ一周, 最高仰角
```
