## 1. プロジェクト概要

### パッケージ名
`ComfyUI-RogoAI-NanoBanana`

### 概念
**4D タイムラプス動画生成システム**

```
時間軸 (t): 被写体のライフステージ（卵→幼虫→蛹→成虫 等）
水平軸 (h): カメラ水平角（0°=正面、360°=一周）
仰角軸 (v): カメラ仰角（-30°〜+60°、7段階）
```

3次元の (t, h, v) 格子を自由に飛び回るパスを定義し、
LTX-Video 2 (AI動画生成) で格子点間を補間して、
「時間 × 空間」を縦横無尽に移動する動画を生成する。

## 6. 重要な実装注意点

### ComfyUI カスタムノードのimport規則

```python
# ✅ 正しい (relative import)
from ._nb4d_paths import SWEEP_PATHS

# ❌ 間違い (absolute import — ComfyUIでは失敗する)
from nodes._nb4d_paths import SWEEP_PATHS
```

### LTX-2 デッドロック問題（解決済み）

ComfyUI の `prompt_worker` スレッドは1つ。
ノード内からAPIで内部ジョブをキューすると永久デッドロックになる。
**V2以降は直接Pythonクラス呼び出し方式で解決済み**。

### モデルファイル名（LTX-2）

```
models/
  ltx2/ltx-2-19b-distilled_Q6_K.gguf
  ltx2/LTX2_video_vae_bf16.safetensors
  ltx2/gemma_3_12B_it_fp4_mixed.safetensors
  ltx2/ltx-2-19b-embeddings_connector_distill_bf16.safetensors
```

### Python変更後はComfyUI再起動必須

カスタムノードのPythonコードを変更しても、ComfyUIの再起動なしには反映されない。
