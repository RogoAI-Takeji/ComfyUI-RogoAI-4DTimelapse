### やること

`ltxv_stage_interpolate.py` の内容を ComfyUI カスタムノード化する。

**作成するファイル**: `nodes/ltxv_interpolator.py`  
**ノード名**: `NB4D_LTXVStageInterpolator`  
**カテゴリ**: `NanoBanana/4DGrid`

---

### 実装方式

ComfyUI ノード内から **同じ ComfyUI の REST API（localhost:8188）** に `queue_prompt` を投げる方式。
LTX-Video 2 の重いモデルを ComfyUI 側で共有できるため VRAM 効率が良い。

```
[NB4D_LTXVStageInterpolator ノード実行]
    ↓
POST http://127.0.0.1:8188/upload/image  （各stageの画像をアップロード）
POST http://127.0.0.1:8188/prompt        （LTXVワークフローをキューに投入）
GET  http://127.0.0.1:8188/history/{id}  （完了をポーリング待機）
    ↓
RGBA PNG連番を output_dir に保存
```

### build_workflow() の構成（ltxv_stage_interpolate.py から移植）

```
ノード "0": UnetLoaderGGUF     → MODEL
ノード "1": DualCLIPLoader     → CLIP  (type="ltxv", clip1=gemma_fp4, clip2=embed_connector)
ノード "2": VAELoader          → VAE
ノード "3": CLIPTextEncode(+)  → CONDITIONING_pos
ノード "4": CLIPTextEncode(-)  → CONDITIONING_neg
ノード "5": LTXVConditioning   → CONDITIONING_pos2, CONDITIONING_neg2
ノード "6": LoadImage          → IMAGE  ← upload_image() で登録した画像
ノード "7": LTXVImgToVideo     → CONDITIONING_pos3, CONDITIONING_neg3, LATENT
             width=768, height=512, length=clip_frames, strength=1.0
ノード "8": CFGGuider          → GUIDER  (cfg=cfg)
ノード "9": KSamplerSelect     → SAMPLER (sampler_name="euler")
ノード "10": LTXVScheduler    → SIGMAS  (steps, max_shift=2.05, base_shift=0.95)
ノード "11": RandomNoise      → NOISE   (noise_seed=seed+stage_idx)
ノード "12": SamplerCustomAdvanced → LATENT_out
ノード "13": VAEDecode         → IMAGE_out
ノード "14": SaveImage         → (出力受け取り用)
```

### `interpolate()` メソッドのロジック

```python
def interpolate(self, keyframes_dir, positive_prompt, negative_prompt, clip_frames,
                angle, angle_start, angle_end, cfg, steps, seed, fps,
                zoom_start, zoom_frames, alpha_thresh, output_dir,
                comfyui_url, model_gguf, model_vae, gemma_fp4, embed_connector):

    # 1. grid_meta.json を読み込む
    kf_dir = Path(keyframes_dir)
    with open(kf_dir / "grid_meta.json") as f:
        meta = json.load(f)
    n_stages = meta["n_stages"]
    n_angles = meta.get("grid_theta", 24)
    bg_gray  = meta.get("render_params", {}).get("bg_gray", 0.12)

    # 2. 出力ディレクトリ作成
    out_dir = Path(output_dir)
    ts = datetime.now().strftime("%Y%m%d_%H%M%S")
    png_dir = out_dir / f"ltxv_angle{angle:03d}_{ts}"
    png_dir.mkdir(parents=True, exist_ok=True)

    # 3. ComfyUI 接続確認
    _api(comfyui_url, "/system_stats")

    # 4. stage ループ
    all_frames = []
    for stage_idx in range(n_stages):
        # 角度計算（スイープ or 固定）
        if angle_start >= 0 and angle_end >= 0:
            t = stage_idx / max(1, n_stages - 1)
            cur_angle = round(angle_start + (angle_end - angle_start) * t) % n_angles
        else:
            cur_angle = angle % n_angles

        # 画像取得・リサイズ
        src = kf_dir / f"stage_{stage_idx:02d}" / f"angle_{cur_angle:03d}.png"
        img = Image.open(src).convert("RGB").resize((768, 512), Image.LANCZOS)
        tmp_png = tmp_dir / f"stage_{stage_idx:02d}.png"
        img.save(tmp_png)

        # アップロード → キュー投入 → 待機 → フレーム取得
        comfy_name = upload_image(comfyui_url, tmp_png)
        wf = build_workflow(comfyui_url, comfy_name, positive_prompt, negative_prompt,
                            clip_frames, cfg, steps, seed + stage_idx,
                            model_gguf, model_vae, gemma_fp4, embed_connector)
        prompt_id = queue_prompt(comfyui_url, wf)
        hist = wait_for_completion(comfyui_url, prompt_id, timeout=600)
        frames = fetch_output_images(comfyui_url, hist)

        # クリップ連結（先頭以外は overlap=1 を除く）
        if stage_idx == 0:
            all_frames.extend(frames)
        else:
            all_frames.extend(frames[1:])

    # 5. RGBA PNG 保存（アルファ抽出 + ズーム）
    for i, pil_img in enumerate(all_frames):
        rgb   = np.array(pil_img.convert("RGB"), dtype=np.uint8)
        alpha = extract_alpha(rgb, bg_gray, alpha_thresh)
        rgba  = np.dstack([rgb, alpha]).astype(np.uint8)
        z = zoom_factor(i, zoom_start, zoom_frames)
        if z > 1.001:
            rgba = apply_zoom(rgba, z)
        Image.fromarray(rgba, "RGBA").save(png_dir / f"frame_{i:05d}.png")

    status = f"完了: {len(all_frames)}フレーム → {png_dir}"
    return (str(png_dir), len(all_frames), status)
```

### ヘルパー関数（ltxv_stage_interpolate.py からそのまま移植）

| 関数 | 場所 |
|------|------|
| `_api(url, path, data)` | HTTP GET/POST |
| `upload_image(url, img_path)` | multipart POST /upload/image |
| `build_workflow(...)` | ワークフロー JSON 構築 |
| `queue_prompt(url, workflow)` | POST /prompt |
| `wait_for_completion(url, prompt_id, timeout)` | /history ポーリング |
| `fetch_output_images(url, history_entry)` | 画像ダウンロード |
| `extract_alpha(rgb, bg_gray, thresh)` | グレー背景からアルファ抽出 |
| `apply_zoom(rgba, zoom)` | ズーム適用 |
| `zoom_factor(i, zoom_start, zoom_frames)` | ズーム係数計算 |
